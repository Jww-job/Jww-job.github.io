JS和ES6语法的使用和区别

js 和 ES6语法的使用和区别

变量声明

  1. js中有三种声明方式 var let const
  2. var为全局作用域，如果有if语句，使用var声明变量，在if语句外，有相同的变量会造成冲突
  3. let和const的区别在于，let可以重新赋值,而const不能进行重新赋值，会报错“常量不能当变量赋值”
  4. 当需要重新赋值的时候，使用let，不需要重新赋值时可以使用const，使用const需要赋予初始值

数据类型

  1. String。Numbers，Boolean，null，undefined
  2. js中没有设置浮点数,小数和整数在判断类型时都为number
  3. 传统字符串拼接采用 + 的方法来进行，较为复杂，在ES6中使用模板字符串来进行拼接，使用${}的方式

字符串的属性和方法

  1. 字符串的数量 length
  2. 将字符串改变为大写 touppercase
  3. 截取字符串 substring(0,5);
  4. 将字符串分割到数组中 split(‘ ，’),中间可以添加“ ，”分割;

Arrays

  1. 使用/* 内容 */ 实现多行注释
  2. 在js中数组可以保存不同类型的值,可以包含多种变量值
  3. 创建新数组 const arr = new Array(1,2,3,4,5); 或者 const arr = [‘apple’,’tomoto’,’banana’]
  4. 在数组中新增值 arr1[3] = ‘tudou’,
  5. typescript中新增了静态输入
  6. 在数据末尾添加数据使用push，在开头加入数据使用unshift，去掉最后一项使用pop，判断是否为数组可以使用Array.isArray(),获取某个数据的索引值使用indexof();

对象语法（键值对）

  1. 使用JSON.Stringfy()方法将键值对转化为json数据

  2. for循环来循环数组

  3. for…of循环数据(查看数据时不需要添加索引值)

  4. foreach,map,filter(foreach和map都是迭代器，filter是选择器)

    使用map会返回一个数值，可以定义变量

  5. ==和===的区别，==不关注数据类型

  6. 三元操作符

函数

  1. ES6箭头函数
  2. 面向对象，可以通过构造函数来创建对象,原型链或者ES6的类来给对象添加方法
    通过构造函数来实例化对象

DOM操作

  1. 选择HTML元素,推荐使用queryselector
  2. 遍历元素
  3. 回收元素
  4. 事件
  5. 表单提交获取
