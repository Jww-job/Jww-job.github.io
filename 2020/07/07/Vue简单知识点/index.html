Vue简单知识点

前端
发布日期:   2020-07-07

VUE系统学习

vue的特点

  1. 使用npm install -g vue-cli安装脚手架
  2. vue –version 或者 vue -v查看版本以及是否安装成功

vue的工作模式

  1. vue区别于传统的编程方式，是一种声明式编程，完全的数据与页面分离的开发
  2. 在我们进行vue开发的时候，首先是创建了一个vue对象
  3. 创建vue对象的时候，传入了一些options：{}，这个{}中包括了el：这个属性决定了vue挂载到哪个元素上，data：包含了我们需要的数据

vue的数据

  1. 拿到数据时，使用v-for来进行数据处理

    <div id="app">
     <ul>
         <li v-for="item in lists">{{item}}</li>
     </ul>
 </div>
 <script>
     const app = new Vue({
         el : '#app',
         data : {
             message: 'my list',
             lists: ['香蕉','苹果','梨子','榴莲']
         }
     })
 </script>

  2. 通过v-on来监听事件，也可以使用@click方式，在options中，methods提供方法的创建

    <div id="app">
     <h2>当前计数为:{{math}}</h2>
     <!-- <button v-on:click='math++'>+</button>
     <button v-on:click='math--'>_</button> -->
     <button @click='add'>+</button>
     <button @click='dec'>_</button>
 </div>

 <script>
     const app = new Vue({
         el : '#app',
         data : {
             math: 0
         },
         methods: {
             add: function(){
                 this.math++
             },
             dec: function(){
                 const math1 = this.math;
                 if(math1 > 0){
                     this.math--
                 }else{
                     this.math == 0
                 }

             }
         }
     })
 </script>

  5. mustache语法，语法中不仅可以直接编写变量，也可以写简单的表达式

  6. v-once语法，当不想随意改变一个数据时，添加v-once，一般不使用

  7. v-text语法，用于改变文本，不建议使用，如果加上mustache语法的话，会覆盖掉改变的文本

  8. v-pre语法：将子元素显示出来，不进行解析

  9. 插值闪烁，在vue2.6版本已经解决，低版本可以使用v-cloak，vue解析之前，div中有一个属性v-cloak，解析之后没有v-cloak

vue的计算属性computed

  1. computed属性存在缓存，不需要再每次调用的时候都进行重新计算。
  2. get和set方法，一般不使用set方法进行设置，只使用get方法为只读属性
  3. 一些小问题：在进行切换用户登录方式的时候，如果输入了内容，切换后会将输入的内容带过来。解析：因为在vue中，渲染虚拟dom时，会尽可能的复用已经存在的元素，而不是重新创建。 解决办法：在input中加入key=””值，加上标签，就不会进行复用
  4. v-if和V-show：v-if和v-show都能决定一个元素是否进行渲染，不过当v-if为false时，根本不会有元素出现在dom中，当v-show为false时，相当于将元素设置了一个display：none。 当我们要在现实和隐藏之间切换频繁时，使用v-show，当只有一次切换的时候，可以使用v-if。
  5. 使用v-for遍历对象时，如果只是获取一个值，那么获取的是value值
    <li v-for = "(value,key,index) in arr">{{key}} {{value}}</li>
  6. 在使用vue中的过滤器filters时，会出现局部过滤器没有效果且报错的情况。这时我们可以在实例外定义一个全局的过滤器，这样就可以产生效果了
//定义全局的过滤器
Vue.filter('showPrice',function(price) {
    return "¥" + price.toFixed(2);
});

//在html代码中使用过滤器
<td>{{item.price | showPrice}}</td>
  1. JavaScript高阶函数 filter/map/reduce
  • filter的回调函数必须为一个boolean值
  • 当返回为true时，函数内部会自动将这次回调的n加入到新的数组中
  • 当返回false时，函数内部会过滤掉这次的n
//filter的使用
const nums = [22,333,12,1412,222,543,15];

let newNums = nums.filter(function(n) {
    return n > 100;//条件判断，为布尔值
});

console.log(newNums);
  • map函数的使用
// map的使用
let new2Nums = nums.map(function(n) {
    return n*2;
})

console.log(new2Nums);
  • reduce的使用：对数组中的所有内容进行汇总
// reduce的使用
let total = newNums.reduce(function(preValue,n){
    return preValue + n;
},0)

console.log(total);
  • 综合写法
let totalMath = nums.filter(function(n) {
    return n<100;
}).map(function(n) {
    return n *2;
}).reduce(function(preValue,n){
    return preValue + n;
},0);

console.log(totalMath);

// 终极写法
let tatal2 = nums.filter(n => n < 100).map(n => n * 2).reduce((pre,n) => pre + n);

console.log(tatal2);

  1. 修饰符

  • v-model.lazy 让数据在失去焦点或者回车的时候才发生改变

  • v-model.number 默认情况下输入的是string类型,使用number修饰符可以直接得到number类型

  • v-model.trim 自动去掉输入的空格，但是只是去掉收尾输入的空格

    <div id="app">
      <input type="value" name="" v-model.lazy="message">
      <h2>{{message}}</h2>

      <input type="value" name="" v-model.number.lazy="age">
      <h2>{{age}}</h2>

      <input type="value" name="" v-model.trim.lazy="name">
      <h2>{{name}}</h2>
  </div>
  <script>
      const app = new Vue({
          el : '#app',
          data : {
              message: 'my list',
              age : 0,
              name:''
          },
      })
  </script>

  2. 父子组件通信 父传子props

  • 子组件不能引用父组件或Vue实例的数据的

  • 通过props像子组件传递数据，通过子组件像父组件发送消息

  • 子组件传父组件 $emit event

  • props的值有两种类型

    • 字符串数组，数组中的字符串就是传递时的名称

    • 对象，对象可以设置传递时的类型，也可以设置默认值

      <div id="app">
  <cpn :cmovies = 'movies' :cmessage = 'message'></cpn>
</div>
<template id="cpn">
  <div>
      <ul>
          <li v-for = 'item in cmovies'>{{item}} </li>
      </ul>
      <h2>{{cmessage}}</h2>
  </div>
</template>
<script>
  // 父传子使用props
  const cpn = {
      template : '#cpn',
      // props : ['cmovies','cmessage'] //数组形式，还可以为对象形式
      props: {
          //类型限制
          // cmovies : Array, //要求传入的数据类型
          //设置默认值

          cmessage : {
              type : String,
              default : '无参数',
              required : true //设置了这个值之后，就变成了必传值
          },
          //类型是一个数组或者对象时，默认值必须是一个函数
          cmovies : {
              type : Array,
              default(){
                  return []
              }
          },
      }
  }
  const app = new Vue({
      el : '#app',
      data : {
          message: 'my list',
          movies : ['sss','dasad','wwwwwwwda']
      },
      components : {
          cpn
      }
  })
</script>
